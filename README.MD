# ScienceHub.ir 


for run this project first install Docker & docker-compose then

```
$ docker-compose up
$ cd backend
$ python3 venv env
$ source env/bin/activate
(evn)$ pip install -r requirements.txt
(env)$ python3 manage.py runserver
```

then head to frontend dir and run `yarn start` to run front end


Please report bugs.
